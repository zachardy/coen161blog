Style Guide:

  • use {...}'s around the text you would like to style, replacing the ... with one of the style tags shown below
  • available style tags (format is "tag" - description):
  • b - bold
  • i - italic
  • 1 - largest text
  • 2 - large text
  • 3 - big text
  • 4 - small text
  • 5 - tiny text
  • 6 - tiniest text
  • example, to bold the phrase "hello world", use style tags as shown below
  • {b}hello world{b}
  • have fun!
